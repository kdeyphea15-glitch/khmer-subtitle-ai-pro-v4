# Khmer Subtitle AI Pro V4 Client

React + Vite + TypeScript frontend for Khmer Subtitle AI Pro V4.

## Features
- Luxury cinematic black-and-gold interface
- One-click Khmer translation and dubbing trigger
- Drag-and-drop video upload panel
- Voice controls and Khmer voice preview
- Animated AI workflow progress
- Video, subtitle, and audio preview

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
